# Code of Conduct

We want Axocoatl to be a welcoming, productive place to build software.

## What we expect

- Be respectful. Disagree with ideas, not people.
- Assume good faith. Most disagreements are misunderstandings.
- Keep discussion technical, on-topic, and constructive.
- Make space for new contributors. Explain context, link docs, share why
  things are the way they are.

## What is not OK

- Personal attacks, demeaning language, or sustained disruption of
  discussion.
- Sharing other people's private information without permission.
- Any conduct that would reasonably make a contributor feel unwelcome or
  unsafe.

## Scope

This applies to all project spaces — GitHub issues, pull requests,
discussions, and any official communication channel.

## Enforcement

Report concerns to **conduct@axocoatl.ai**. Reports stay confidential.
Maintainers will respond and decide on appropriate action, which can range
from a private note to removal from the project.

We aim to be fair, fast, and direct.
